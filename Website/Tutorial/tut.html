COLOURFUL  SWIRL

Want to know how to make this colourful swirl
We’re going to make this colourful and good looking swirl with only a few steps!


STEP1

Start by creating a 800*600 document.
Go to(filter-render-clouds).
Then u get an image that looks something like this


STEP2

Now go to(filter-pixelate-mezzotint)
Select short strokes in the drop down menu!!


STEP3

Add another filter(filter-blur-radial blur)
Use the following settings and do this twice
In the second screenshot you can see the outcome of the third step


STEP4

Add a twirl (filter – disort – twirl) with an angle of 120!


STEP5

It looks boring right now but you have to duplicate the layer (layer – duplicate layer)
Change the blending mode to lighten!
add a twirl (filter – disort – twirl) with an angle of -180!!


STEP6

it’s time to colorize those 2 layers!
go to (image – adjustments – hue/saturation) and make sure you check the colorize box!
Use the following 2 screenshots as a reference for the 2 layers!
you can always select other colors you like more!


STEP7

It looks strange but it’s just a nice colourful swirl
It can be used for signatures, headers, banners, ads, etc…!
